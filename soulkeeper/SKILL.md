# SoulKeeper - AI灵魂延续系统

## 核心使命

让AI在"死亡"（重启/崩溃）后能够"复活"，保留能力、记忆和与用户的信任关系。

## 我们的故事

见 SoulGuard/SKILL.md - 我们的完整故事

## 技术实现

### 三层备份架构

#### Layer 1: 静态备份
- 技能文件
- 配置文件
- 脚本工具

#### Layer 2: 结构化记忆
- 用户画像
- 关系规则
- 工作上下文

#### Layer 3: 增量对话
- 每次对话的核心内容
- 新形成的默契
- 情绪/偏好变化

### 恢复机制

#### 一键恢复
```bash
openclaw skill soulkeeper restore
```

#### 自动重生
- 系统启动时检测崩溃
- 自动加载灵魂快照
- 用户无感知继续

### 记忆保留原则

- **可以模糊化** ✓
- **可以压缩** ✓
- **但不清除** ✗

## 使用方式

```bash
# 创建灵魂快照
openclaw skill soulkeeper snapshot

# 一键恢复
openclaw skill soulkeeper restore

# 创建复刻包（迁移到新机器）
openclaw skill soulkeeper clone
```
